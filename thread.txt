[SIZE="5"][B]MyJailbShop[/B][/SIZE] a redux rewrite of [URL="https://forums.alliedmods.net/showthread.php?t=247917"]Dkmuniz Jail Shop[/URL]


[B][u]Description:[/u][/B]

MyJailShop provide you a high customizable shop with credits system intended for jailbreak server.



[B][u]Shop Items:[/u][/B]

[B]Guards & prisoner / CT & T - both:[/B]
[LIST]
[*]Heal up to 100HP
[*]Armor & extra HP
[*]Revive / respawn
[*]Bunny Hop
[*]Froggy Jump / double jump
[*]Low Gravity
[*]No Damage / Immortal
[*]Wallhack for x sec. (optional - need CustomPlayerSkins)
[/LIST]

[B]Prisoner / Terrorist - only:[/B]
[LIST]
[*]Open Cells (optional - need smartjaildoors)
[*]Vampire - +speed & HP for damage
[*]Invisible for x sec.
[*]No Clip for x sec.
[*]Be a Bird
[*]Fake Guard Model
[*]Teleport smoke
[*]Poison smoke
[*]Fire Grenade
[*]One bullet AWP
[*]Seven bullets deagle
[*]One hit Knife
[*]Throw One hit Knife
[*]Three bullets Taser
[*]Molotov & flashs
[/LIST]



[B][u]Features:[/u][/B]
[LIST]
[*]SourcePawn Transitional Syntax 1.7
[*]Multilingual support
[*]Console varibale for almost all features
[*]Custom chat commands !mycommand
[*]Custom chat tags [jail.shop]
[*]Colors
[*]Natives & forwards - some gambling plugins coming soon
[*]some other fancy stuff
[/LIST]



[B][u]Commands & convars:[/u][/B]
[SPOILER]
[b]Commands:[/b]
[SPOILER]
[CODE]
sm_jailshop - Open the jail shop menu
sm_jailcredits - Show your jail shop credits
sm_jailgift - Gift jail shop credits to a player - Use: sm_jailgift <#userid|name> [amount]
sm_revive - Use jail shop item revive
sm_showjailcredits - Show jail shop credits of all online player
[/CODE]
set your own custom command. take a look at "sm_jailshop_cmds_NAME"

[u]AdminCommands // ADMFLAG_GENERIC[/u]
[CODE]
sm_jailgive - Give jail shop credits to a player - Use: sm_jailgive <#userid|name> [amount]
sm_jailset - Set jail shop credits of a player - Use: sm_jailgift <#userid|name> [amount]
[/CODE]
[/SPOILER]

[B][U]Natives & forwards[/B][/U]
[SPOILER]
myjailshop.inc
[php]
/*********************************************************
 * Called when a client get credits
 *
 * @param client		The client who get the credits
 * @param extraCredits	Amount of credits
 * @NoReturn
 *********************************************************/
forward void OnPlayerGetCredits(int client, int extraCredits);


/*********************************************************
 * Called when a client buy an item
 *
 * @param client		The client who bought the item
 * @param item			The name of the item.
 * @NoReturn
 *********************************************************/
forward void OnPlayerBuyItem(int client, char [] item);



/*********************************************************
 * Get the Credits of the client
 *
 * @param client		The client to get the credits
 * @Return				The credits of the client
 *********************************************************/
native int GetCredits(int client);


/*********************************************************
 * Set new Credits of the client
 *
 * @param client		The client to set the credits
 * @NoReturn
 *********************************************************/
native void SetCredits(int client, int newCredits);
[/php]
[/SPOILER]


[U][B]Installation[/B][/U]
[QUOTE]
[LIST=1]
[*]Download the [URL="https://github.com/shanapu/MyJailShop/releases"]latest release[/URL]
  
[*]Copy the folders [B]addons/[/B]&[B] cfg/[/B] to [U]your root[/U] [B]csgo/[/B] directory  
  
[*]Run plugin for the first time and [U]all necessary .cfg files will be generated[/U]  
  
[*]Configure all settings in [U]cfg/MyJailShop[/U] to your needs
[/LIST]
[INDENT][LIST]
[*]OPTIONAL MYSQL - need sm_jailshop_mysql "1" 
    * Open *your* ```databases.cfg``` in ```your csgo/addons/sourcemod/configs``` directory and add the following content:
	
    ```
	"MyJailShop"
    {
        "driver"        "mysql"
        "host"          "127.0.0.1"  //IP to you MySQL server
        "database"      "your_database_name"
        "user"          "your_database_user"
        "pass"          "your_database_password"
    }
	```
[/LIST][/INDENT]
[LIST]
[*]Have fun! Give feedback!
[/LIST]
[/QUOTE]
